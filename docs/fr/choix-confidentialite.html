Vos contrôles

Choix de confidentialité

Vous pouvez contrôler les autorisations, le consentement publicitaire, les abonnements et les données enregistrées localement.

Microphone et notifications

Ouvrez Réglages iOS, choisissez Mimi, puis modifiez l’autorisation souhaitée. Sans microphone, l’enregistrement audio est indisponible, mais les autres parties de l’app restent utilisables.

Confidentialité publicitaire

Lorsque cela est requis, Mimi affiche un message de consentement Google avant de demander des annonces. Si une ligne « Choix de confidentialité » apparaît dans le profil Mimi, touchez-la pour consulter ou retirer les choix gérés par la plateforme de consentement Google.

Des annonces limitées ou non personnalisées peuvent encore être affichées lorsque la loi le permet après un refus de personnalisation.

Données locales

Mimi conserve le profil, les préférences, les enregistrements, l’historique et la progression des quiz sur votre appareil. Utilisez les contrôles disponibles pour supprimer certains contenus. La désinstallation supprime le conteneur local de Mimi.

Abonnements

Gérez ou annulez Mimi+ dans Réglages iOS, sous votre identifiant Apple et Abonnements. Utilisez « Restaurer les achats » dans Mimi pour restaurer un achat éligible.

Demandes

Mimi n’exploite ni compte utilisateur ni base de profils cloud. Pour demander l’accès, la correction ou la suppression de messages d’assistance que nous contrôlons, écrivez à maxime.pietri@icloud.com.
